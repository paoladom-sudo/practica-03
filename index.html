Sobre mí

Habilidades

  • Diseño gráfico

    • Diseño gráfico: Es la disciplina que se encarga de comunicar ideas y mensajes a través de elementos visuales como colores, tipografías, imágenes y composiciones. Su objetivo es transmitir información de forma clara, atractiva y coherente, fortaleciendo la identidad visual de marcas, productos o proyectos.

  • UX / UI

    • UX/UI:El diseño UX se enfoca en la experiencia del usuario, buscando que la navegación sea sencilla, intuitiva y agradable, mientras que el UI se centra en la parte visual de las interfaces, como botones, menús y estilos gráficos, asegurando que todo sea atractivo y funcional.

  • HTML5, CSS3, JavaScript básico

    • HTML5: Es el lenguaje que permite estructurar el contenido de una página web, organizando textos, imágenes, enlaces y otros elementos para que el navegador los interprete y los muestre correctamente.

    CSS3: Se utiliza para dar estilo y diseño a las páginas web, definiendo colores, tipografías, tamaños, márgenes y la disposición visual de los elementos, haciendo que el sitio tenga una apariencia estética y profesional.

    JavaScript básico: Es un lenguaje de programación que añade interactividad a los sitios web, permitiendo crear acciones dinámicas como botones que responden, formularios interactivos o animaciones simples.

  • Prototipado y animación

    • Prototipado: Consiste en crear versiones preliminares de un diseño o producto digital para visualizar su funcionamiento, estructura y experiencia de usuario antes de su desarrollo final.

    Animación: Aporta movimiento a los elementos visuales, haciendo que las interfaces y piezas gráficas sean más dinámicas, atractivas y expresivas, mejorando la experiencia y captando la atención del usuario.

Proyectos

Diseño de Identidad

Este proyecto se centra en el desarrollo de una identidad visual sólida y coherente para una marca moderna. La propuesta incluye la creación del logotipo, selección de tipografías y definición de una paleta cromática que transmite profesionalismo, creatividad y confianza.

El diseño busca reflejar los valores de la marca y fortalecer su presencia visual, garantizando una aplicación consistente en distintos soportes como papelería corporativa, material promocional y entornos digitales.

Animación de Presentación

Este proyecto consiste en una animación dinámica creada para presentar de forma visual e impactante los valores y servicios de una marca. A través de transiciones suaves, ritmo visual equilibrado y elementos gráficos modernos, el video busca captar la atención del espectador y transmitir el mensaje de manera clara y profesional.

La animación está pensada para su uso en presentaciones digitales, redes sociales y material promocional, reforzando la identidad visual y mejorando la comunicación con el público objetivo.
